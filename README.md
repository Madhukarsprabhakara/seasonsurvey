# Seasonsurvey

### About Seasonsurvey

Open-source survey platform focussed on collecting and analyzing qualitative data.

### Tech stack

- Laravel
- Tailwind
- Vue 3
- Inertia 1.0
- PostreSQL DB
- Docker
- Llama 3.1 8b
